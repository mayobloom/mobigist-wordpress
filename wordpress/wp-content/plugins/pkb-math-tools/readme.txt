=== PKB Math Tools ===

PKB Math Tools adds an Extended Math block for MathLive-based LaTeX writing in
the WordPress block editor.

It keeps the default Math block intact. The Extended Math block provides a
MathLive editing surface with structured math input, command suggestions,
inline/line mode switching, clear, undo, redo, done controls, and a resizable
editing surface.

The plugin also adds a Math Tools panel to the block settings sidebar. The
panel provides quick LaTeX snippet buttons for common mathematical structures,
linear algebra, probability, optimization, machine learning, Greek letters,
symbols, and selected example formulas.

The Math Tools sidebar inserts snippets at the current MathLive cursor
position.

Bundled third-party library:
- MathLive, included under assets/vendor/mathlive. See the bundled LICENSE.txt.

=== PKB Math Tools ===

PKB Math Tools is a small editor add-on for the WordPress core Math block.

It keeps the default Math block intact and adds a Math Tools panel to the block
settings sidebar. The panel provides quick LaTeX snippet buttons for common
mathematical structures, linear algebra, probability, optimization, machine
learning, Greek letters, symbols, and selected example formulas.

The plugin also adjusts the Math block editor popover so the LaTeX input field
can be resized both vertically and horizontally while writing longer formulas.
